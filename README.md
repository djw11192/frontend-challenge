# Take home Test

## Refactor Test
Steps:
1. Clone the repository
2. cd into frontend-challenge
3. Run `npm install`
4. Run `node node-refactor.js` to make sure output is correct with refactored code.

## PetList Test
Steps:
1. If you have cloned repository, cd into frontend-challenge/angular-petlist
2. Run `npm install`
3. Run `node index.js` or `nodemon`
4. See results in browser at localhost:3000
